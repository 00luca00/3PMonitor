# 3PMonitor
Simple terminal CPU/RAM monitor for Linux/Windows/MacOS

Requirements:
- Python 3.8+
- psutil
- curses

Install (system-wide): <br>
sudo mv 3pmonitor /usr/local/bin/3pmonitor <br>
sudo chown root:root /usr/local/bin/3pmonitor <br>
sudo chmod 755 /usr/local/bin/3pmonitor <br>
sudo pip3 install psutil <br>
